- Index.tsx - Strona główna

### /src/services/ - Usługi i API:
- authService.ts - Autoryzacja

- petService.ts - Obsługa zwierząt

### /src/integrations/supabase/ - Integracja z bazą danych:

### /src/contexts/ - React Context:
- AuthContext.tsx - Kontekst autoryzacji

### /src/types/ - Definicje typów TypeScript:
- index.ts - Główne typy
- client.ts, pet.ts, visit.ts - Typy danych
- subscription.ts - Typy subskrypcji

## GŁÓWNE FUNKCJONALNOŚCI:

1. **SYSTEM AUTORYZACJI** - Logowanie, rejestracja, role użytkowników
2. **ZARZĄDZANIE KLIENTAMI** - CRUD klientów z formularzami
3. **ZARZĄDZANIE ZWIERZĘTAMI** - Profile zwierząt, historia medyczna
4. **SYSTEM WIZYT** - Planowanie i zarządzanie wizytami
5. **PROFILE SPECJALISTÓW** - Prezentacja specjalistów z specjalizacjami
6. **SYSTEM SUBSKRYPCJI** - Pakiety płatności i limity użytkowania
7. **PANEL ADMINISTRATORA** - Zarządzanie całym systemem
8. **KARUZELA SPECJALISTÓW** - Prezentacja zweryfikowanych specjalistów
9. **KATALOG** - Przeglądanie i filtrowanie specjalistów
10. **NOTATKI ZWIERZĄT** - System notatek z załącznikami

## BAZA DANYCH (Supabase):

## TECHNOLOGIE:
- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- React Router Dom - routing
- Supabase - backend i baza danych
- React Query - zarządzanie stanem
- React Hook Form - formularze
- i18next - internacjonalizacja
- Embla Carousel - karuzele
